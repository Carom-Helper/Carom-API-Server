balls_coord_detect

URL Patterns/api/detection/balls-coord/ <int:carom_id>/<str:usr>
Done
MethodGETPUT

Description

carom 이미지 ID를 입력하면,

carom 이미지의 당구공 좌표를 찾아서

공 좌표를 가져간다. 이미 찾아졌다면, 바로 가져간다.

정사영이 필요한 이미지가 입력값으로 들어온다.
